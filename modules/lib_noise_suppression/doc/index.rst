Noise Suppression Library
=========================

Introduction
************

``lib_noise_suppression`` is a library of functions for performing Noise Suppression on input data.

On GitHub
---------

``lib_noise_suppression`` is present as part of ``sw_avona``. Get the latest version of ``sw_avona`` from
``https://github.com/xmos/sw_avona``. ``lib_noise_suppression`` is present within the `modules/lib_noise_suppression` directory in ``sw_avona``.

API
---

To use the functions in this library in an application, include :ref:`sup_api_h` in the application source file.
